# What is _discovery_?

Discovery is a simple standalone program to find your computers in your LAN/VPN
or whatever network.

# Compilation

## Generating Makefiles

Only run once:

$ qmake

## Generating Languages Files

Only needs to be run if the language files changed:

$ lrelease discovery.pro

## Building

Run always:

$ make
